## Typography — Type Scale

M3 Type Scale は5カテゴリ x 3サイズの15段階で構成。Display/Headline は Regular (400)、Title/Label は Medium (500)、Body は Regular (400)。

### Token Reference

`text-*` 1クラスで font-size + line-height + letter-spacing + font-weight が全て自動適用。

| Class              | size | height | spacing | weight |
| ------------------ | ---- | ------ | ------- | ------ |
| `text-display-lg`  | 57px | 64px   | -0.25px | 400    |
| `text-display-md`  | 45px | 52px   | 0px     | 400    |
| `text-display-sm`  | 36px | 44px   | 0px     | 400    |
| `text-headline-lg` | 32px | 40px   | 0px     | 400    |
| `text-headline-md` | 28px | 36px   | 0px     | 400    |
| `text-headline-sm` | 24px | 32px   | 0px     | 400    |
| `text-title-lg`    | 22px | 28px   | 0px     | 500    |
| `text-title-md`    | 16px | 24px   | 0.15px  | 500    |
| `text-title-sm`    | 14px | 20px   | 0.1px   | 500    |
| `text-body-lg`     | 16px | 24px   | 0.5px   | 400    |
| `text-body-md`     | 14px | 20px   | 0.25px  | 400    |
| `text-body-sm`     | 12px | 16px   | 0.4px   | 400    |

## Component Token Mapping

コンポーネントで使用しているセマンティックトークンとM3ロールの対応表。

| Token                    | M3 Role                  | Tailwind Class                |
| ------------------------ | ------------------------ | ----------------------------- |
| `background`             | `surface`                | `bg-background`               |
| `foreground`             | `on-surface`             | `text-foreground`             |
| `primary`                | `primary`                | `bg-primary`                  |
| `primary-foreground`     | `on-primary`             | `text-primary-foreground`     |
| `secondary`              | `secondary-container`    | `bg-secondary`                |
| `secondary-foreground`   | `on-secondary-container` | `text-secondary-foreground`   |
| `destructive`            | `error`                  | `bg-destructive`              |
| `destructive-foreground` | `on-error`               | `text-destructive-foreground` |
| `card`                   | `surface-container-low`  | `bg-card`                     |
| `accent`                 | `secondary-container`    | `bg-accent`                   |
| `muted`                  | `surface-container`      | `bg-muted`                    |
| `muted-foreground`       | `on-surface-variant`     | `text-muted-foreground`       |
| `input` / `border`       | `outline-variant`        | `border-input`                |
| `ring`                   | `primary`                | `ring-ring`                   |
