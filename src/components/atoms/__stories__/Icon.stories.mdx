# 使い方

- `src/atoms/icons`ディレクトリにアイコン用`svg`ファイルを配置し`Icon.tsx`で svg を import しコンポーネントを作成する
- ※ `@svgr/webpack`でインライン化している

# サイズ

size prop にサイズ（`number | string`）を入力する（デフォルトは `24px`）

```
<InformationIcon size="32" />
```

# カラー

color prop にカラーコード(`string`)を入力する（デフォルトは `currentColor`）
例

```
<InformationIcon color="#1EA7FD" />
```

# 回転

rotate prop に CSS の`transform: rotate(n)`に相当する値(`string`)を入力する（デフォルトは`undefined`）

```
<ChevronIcon rotate="180deg" />
```
